ITEM - Playwright
DESCRIPCION - Playwright
UNIDAD DE MEDIDA - USUARIO 
CANTIDAD - Playwright
MARCA - USUARIO
OBS USUARIO - USUARIO 
CONVERSIÓN USD - USUARIO(cuantos pesos equivale un dólar)
COSTO UNIT USD - FORMULA -> COSTO FINAL UNITARIO ARS / CONVERSIÓN USD 
COSTO TOTAL USD - FORMULA -> COSTO FINAL TOTAL ARS / CONVERSIÓN USD
COSTO FINAL UNITARIO ARS - USUARIO O CALCULO - FORMULA -> COSTO FINAL TOTAL ARS / CANTIDAD
COSTO FINAL TOTAL ARS - USUARIO o CALCULO - FORMULA -> COSTO FINAL UNITARIO ARS * CANTIDAD
RENTA MINIMA ACEPTABLE - USUARIO( usuario elige que utilidad marcarle 1.3 seria una utilidad 30%)
PRECIO UNIT ACEPTABLE- FORMULA -> COSTO FINAL UNITARIO ARS  * RENTA MINIMA ACEPTABLE
PRECIO TOTAL ACEPTABLE - FORMULA -> COSTO FINAL TOTAL ARS * RENTA MINIMA ACEPTABLE
PRECIO DE REFERENCIA - Playwright
PRECIO DE REFERENCIA UNITARIO - FORMULA -> PRECIO DE REFERENCIA / CANTIDAD
RENTA REFERENCIA - FORMULA -> (PRECIO DE REFERENCIA / COSTO FINAL TOTAL ARS)-1 ( representar como porcentaje)
MEJOR OFERTA ACTUAL -Playwright
OFERTA PARA MEJORAR -Playwright ( dato clave)
PRECIO UNITARIO MEJORA - FORMULA -> OFERTA PARA MEJORAR / CANTIDAD
OFERTA PARA MEJORAR -Playwright ( dato clave)
RENTA PARA MEJORAR - FORMULA -> (OFERTA PARA MEJORAR/ COSTO FINAL TOTAL ARS )-1 ( representar como porcentaje)

    y viceversa  si el usuario ingresa COSTO FINAL TOTAL ARS apartir de este calcular COSTO FINAL UNITARIO ARS
